# Azure Configuration Drift Detection

This repository contains custom Azure Policies designed to detect and prevent
cloud configuration drift and security misconfigurations.

## Project Scope
- Detect public storage exposure
- Enforce NSG security rules
- Restrict excessive RBAC permissions

## Technologies
- Azure Policy
- Azure RBAC
- Network Security Groups

## Research Context
This repository supports a master thesis on Cloud Security Posture Management (CSPM).
